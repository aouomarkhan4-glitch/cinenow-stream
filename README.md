CineNow: Streaming Service Template
Overview

A fully responsive, static front-end template for a video streaming platform, built using pure HTML and CSS. Designed with a cinematic black-and-red theme, CineNow provides a modern UI and complete user flow pages for streaming services.

Features
Dark & Modern UI

Sleek black-and-red theme.

Cinematic layout and smooth styling.

Complete Page Flow

Includes all the essential screens for a basic streaming platform:
